
# Gettings Started
- Move the root folder 
- Run `yarn install` for installing the dependencies.
- Run `yarn start` and open the url `localhost:3000` in your browser.
- That's it.


# Technology stack

- ReactJS
- SCSS
- RESTful API with hacker API
